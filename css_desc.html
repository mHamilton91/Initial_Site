Cascading Style Sheet

CSS is used to modify how the HTML is displayed without changing the HTML

Css can be added to HTML in 3 ways

  • Added to an elements "style" attribute
  • Adding the CSS into a style tag in the HTML
  • Creating a .css file referencing it in the HTML page. Perferred Method.

The Preferred Method

<link href="../styles/stylesheet.css" rel="stylesheet">

CSS Syntax

p {
  color: #333;
  font-size: 30px;
  margin: 20px
}

There are 3 ways to target HTML elements to apply CSS styles

  • HTML Tags (e.g. p, h1, div, code)
  • A class name. Set in the element with the class attribute and referenced using .'class'
  • An ID name. Set in the element with the id attribute and referenced using #'id'

CSS Selector Hierarchy

  1. Tag - Overwrites the default browser styles
  2. Class - Targets more specific elements than the tag selector
  3. ID - The most specific selector. Targets a single element and overwrites everything
  4. Important - Rules marked with !important overwrites everything
