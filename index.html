A div is a containing block, primarily used with CSS to adjust the layout of the web page.

HTML Attributes

Relative Paths

  • filename - looks in current directory
  • folder/filename - starts in current directory and looks in the folder for the specified file
  • ../folder/filename - goes back a directory before looking for the folder and filename
  • /folder/filename - '/' starts in the root directory

Absolute Path

  • http://www.site.com/folder/filename
